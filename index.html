CSS Snippet Cheatsheet Wireframe

A place to store my favorite CSS code snippets.

Flexbow Row

Use these three properties to create a Flexbox layout.


.row {
    display: flex;
    flex-direction: row;
    flex-wrap: wrap;
}

Flexbow Column

Use these two properties to create a Flexbox column layout


.column {
    display: flex;
    flex-direction: column;
}

Flexbow Flow

Use this to shorhand you Flexbox properties


.flow {
    flex: none "flex-grow" "flex-shrink" "flex-basis";
}

Justify content

Use these to align things along the main axis


.justify {
    justify-content: flex-start | flex-end |;
}

Align Items

Use these to align things along the cross axis of the current line.


.align {
    align-items: stretch | flex-start;
}

Align Content

Used to align flex containters.


.acontent {
    align-content: flex-start | flex-end;
}
